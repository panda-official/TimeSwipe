◆ Start()

bool TimeSwipe::Start ( ReadCallback  cb)

Start reading Sensor loop.

Only one instance of TimeSwipe can be running each moment of the time

After each sensor read complete cb called with vector of SensorsData

Buffer is for 1 second data if cb works longer than 1 second, next data can be loosed and next callback called with non-zero errors

Function starts two threads: one thread reads sensor values to the ring buffer, second thread polls ring buffer and calls cb

Function can not be called from callback

Parameters
cb
Returns
false if reading procedure start failed, otherwise true

bool TimeSwipe::StartPWM ( uint8_t  num,
uint32_t  frequency,
uint32_t  high = 4095,
uint32_t  low = 0,
uint32_t  repeats = 0,
float  duty_cycle = 0.5 
)

Start PWM generator Method can be called in any time.

Parameters
num- output number - possible values are 0 or 1
frequency- periods per second - possible values between 1 and 1000
high- PWM signal high value - possible values are 0..4095 high >= low, default is 4095
low- PWM signal low value - possible values are 0..4095 low <= high, default is 0
repeats- number of periods to repeat. PWM generator will work (repeats/frequency) seconds after repeats number get exhausted PWM goes to stop state and StartPWM can be called again 0 is for unlimited repeats (default)
duty_cycle- part of PWM period when signal is in high state. 0.001 duty_cycle <= 0.999. default value is 0.5
Returns
false if at least one wrong parameter given or generator already in start state
